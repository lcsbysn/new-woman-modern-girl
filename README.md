# New Woman · Modern Girl

> ⚠️ **Work in Progress** — diese Website wird laufend erweitert und überarbeitet.

Digitale Begleitausstellung zum Seminar *„Neue Frauen" und „Moderne Mädchen": Weiblichkeitsentwürfe in der chinesischen Republikzeit*. Die Seite stellt Frauen der chinesischen Republikzeit (1912–1949) vor — Revolutionärinnen, Schriftstellerinnen, Aktivistinnen und Medienschaffende — und macht sie sichtbar.

## Inhalte

- **Startseite** – Einführung und interaktive Galerie
- **Galerie** – virtuelle Porträtausstellung
- **Profile** – Biografien, Zeitleisten, Bildergalerien und Quiz
- **Quiz** – Thesen zu Feminismus und Frauengeschichte
- **Sichtbarkeit** – historisches Gedächtnis und Wikipedia-Präsenz
- **Quellen** – Primärquellen und Zitate

## Technik

Statische Website ohne Build-System: reines HTML, CSS und JavaScript. Zum lokalen Ansehen einfach `index.html` im Browser öffnen — kein Server nötig. Zweisprachig (Deutsch / Englisch).
